# smart-expense-savings-manager

A Java console application designed to help users track income, expenses, savings goals, and financial progress

Features

* Add income transactions
* Add expense transactions
* View all transactions
* Calculate current balance
* Track personal finances through a simple console interface

Technologies Used

* Java
* Object-Oriented Programming (OOP)
* ArrayList
* Console-Based User Interface

Project Structure

* Transaction.java
    * Represents a financial transaction.
* FinanceManager.java
    * Manages transactions and balance calculations.
* SmartExpenseSavingsManager.java
    * Main application and user menu.
